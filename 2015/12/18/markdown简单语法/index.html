markdown支持html语法

换行

单一段落用一个空白行

连续两个空格 会变成一个

连续3个符号，然后是空行，表示 hr 横线

标题

生成h1–h6,在文字前面加上 1–6个 # 来实现

######h1

列表

支持嵌套

  • 效果
  1. 声声慢·寻寻觅觅
  2. [宋] 李清照

  • 乍暖还寒时候，最难将息
  • 三杯两盏淡酒，怎敌他、晚来风急？
  • 雁过也，正伤心，却是旧时相识
  • 满地黄花堆积。憔悴损，如今有谁堪摘？
  • 守着窗儿，独自怎生得黑？

表格

文字默认居左

  • 效果
表头 表头 表头
内容 内容 内容
长一点的内容 长一点的内容 长一点的内容

多余空格忽略不计
-为看起来方便 数量可写多个
冒号:在哪侧靠哪侧对齐

超链接

[name](url "超链接title")

Markdown本身语法不支持链接在新页面中打开，可以直接写a标签 target="_blank"
或直接良好习惯 按住command -- 新标签页打开， 按住shift -- 新窗口打开

插入图片

![name](url "图片title")

引用表示法: ![alt_text][id],定义 [id]:url

直接使用<img>，这样可以指定图片的大小尺寸

特殊符号

用\来转义，表示文本中的markdown符号

可以在文本
文本前后各加一个符号，表示斜体种直接使用html标签，但是要注意在使用的时候，前后加上空行
